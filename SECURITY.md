# Security Policy

## Supported versions

Only the latest released version (see [CHANGELOG.md](CHANGELOG.md)) receives fixes.

## Reporting a vulnerability

archcheck is a CLI that parses untrusted source trees, so parser crashes or memory-safety
issues triggered by crafted input are in scope.

Please report vulnerabilities privately via
[GitHub Security Advisories](../../security/advisories/new) ("Report a vulnerability").
Do not open a public issue for exploitable bugs.

You can expect an acknowledgement within a week. Fixes ship as a patch release; the
advisory is published after the fix is out.
