# Code of Conduct

This project follows the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, assume good faith, focus on the technical substance. Harassment,
personal attacks, and trolling are not tolerated.

Instances of unacceptable behavior may be reported to the maintainer via a private
[GitHub security advisory](../../security/advisories/new) or by opening an issue asking
for a private contact channel. All reports will be reviewed and handled confidentially.
