# External OpenWiki practices — adopted method

## Purpose

`status: partial` because this page describes an **external method**, not
archcheck behavior — it is not verifiable against this repo's code the way a
`rule`/`feature` page is. It exists so a future agent knows *why* `docs/openwiki/`
looks the way it does, and can tell "method we borrowed" apart from "archcheck
fact." Do not treat anything below as a claim about archcheck itself.

## Adopted method (summary)

The task that produced this wiki layer (`backlog/new/169_maj_openwiki_agent_context_layer.md`)
was adapted from an external spec built around the OpenWiki idea of a
source-backed, agent-navigable wiki. The parts actually adopted here:

- **Source of truth = code/tests, not the wiki.** Pages are explicitly
  *derived* documentation; when a page disagrees with code, tests, or
  CHANGELOG, the code wins and the page gets fixed (`index.md:30`).
- **Wiki pages are derived, not authored from memory.** Every factual claim
  must cite a repo path or `path:line`; unverifiable claims are marked
  `unverified` and parked, never asserted
  (`backlog/new/169_maj_openwiki_agent_context_layer.md:49-59`).
- **Evidence discipline as a first-class schema field.** `status` +
  `source_paths` + `last_checked_commit` in every page's front matter, so
  staleness is machine-checkable (see `schema.md`, `scripts/check_openwiki.py`).
- **An index page + an append-only log.** `index.md` as the single entry map,
  `log.md` as a running record of what was added/changed and when — rather
  than large auto-generated wiki dumps.

What was explicitly **not** adopted (per the task's "Optional / deferred"
section, `backlog/new/169_maj_openwiki_agent_context_layer.md:131-144`): an
OpenWiki CLI invocation, and any LLM-calling CI workflow
(`openwiki-update.yml`-style). This repo's wiki is hand-authored by an agent
in a normal editing session, not generated by a scheduled external tool call.
Whether an OpenWiki CLI or update-workflow genuinely applies to archcheck was
left unverified — see `backlog/unverified.md`.
